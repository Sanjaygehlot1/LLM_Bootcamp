# Machine Learning & MLOps Learning Journey

This repository documents my journey of learning and building projects in **Machine Learning and MLOps**.

The goal is to understand ML concepts and apply them by building **real-world systems**, including model development, data pipelines, and deployment workflows.

I come from a **software engineering and DevOps background**, so this repository focuses on combining **Machine Learning with production engineering practices**.

## What This Repo Will Contain

* Machine Learning experiments and notebooks
* End-to-end ML projects
* RAG and AI application systems
* MLOps pipelines and infrastructure
* Model deployment and monitoring experiments

## Technologies

* Python
* Scikit-learn
* Pandas / NumPy
* Elasticsearch
* LLM APIs
* Docker & CI/CD (for MLOps)

## Goal

To build strong expertise in **Machine Learning and MLOps** by developing practical systems and documenting the learning process.
